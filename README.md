🗂️ FileVault – Modern File Manager

FileVault is a modern file management application built with Python and Streamlit that allows users to perform file operations through a clean and interactive web interface.

✨ Features

- ➕ Create new files
- 📖 Read file contents
- ✏️ Update existing files
  - Rename files
  - Append content
  - Overwrite content
- 🗑️ Delete files with confirmation
- 📁 Workspace file explorer
- 🎨 Modern dark-themed UI
- ⚡ Fast and lightweight application

---

🛠️ Technologies Used

- Python 3
- Streamlit
- Pathlib

🚀 Installation

1. Clone the Repository

git clone https://github.com/dpkumar2813-tech/FileVault.git
cd FileVault

2. Install Dependencies

pip install -r requirements.txt

Or install Streamlit manually:

pip install streamlit

3. Run the Application

streamlit run app.py

---

🎯 How It Works

Create File

Create a new file by providing a filename and content.

Read File

View the contents of an existing file.

Update File

Modify existing files by:

- Renaming
- Appending content
- Overwriting content

Delete File

Permanently remove files with a confirmation prompt.
